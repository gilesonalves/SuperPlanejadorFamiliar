- sp-textured.ico
- sp-textured-24.png
- sp-textured-48.png
- sp-textured-128.png
- sp-textured-180.png  (Apple touch icon)

- sp-textured-256.png

- sp-textured-1024.png (Splash opcional)
- sp-textured.png
- sp-textured-1600x900.png
- sp-textured-1200x630.png (Open Graph)
- site.webmanifest
- head-snippet.html



Como usar (Vite/React):
1) Copie todos os arquivos para a pasta /public do seu projeto.
2) Em index.html, substitua o <head> pelas tags do arquivo head-snippet.html
3) Faça Ctrl+F5 ou limpe o cache do navegador.
4) Teste o PWA: Instale/Adicionar à tela inicial para verificar os ícones.

Dica: Se o .ico não aparecer no Windows, mantenha também os PNGs 48/192.
